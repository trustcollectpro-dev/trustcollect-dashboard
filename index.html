Comment fonctionne le widget

Le widget TrustCollect est un petit script que vous collez sur votre site web. Il affiche automatiquement vos avis approuvés dans un carrousel élégant, mis à jour en temps réel.

Seuls les avis que vous avez approuvés (statut "Affiché") apparaissent
Design adaptatif — fonctionne sur mobile et desktop
Mise à jour automatique dès qu'un avis est approuvé
Aucune compétence technique requise pour l'installation

Installer le widget

1

Copiez le code d'intégration

Cliquez sur "Copier" pour copier votre code personnalisé avec votre identifiant client.

Chargement…
2

Collez-le sur votre site

Dans l'éditeur de votre site (Webflow, Wix, WordPress…), collez le code juste avant la balise </body> de votre page. Sur la plupart des plateformes, il y a un champ "Custom code" ou "HTML personnalisé" dans les paramètres du site.

3

Approuvez vos avis

Dans la section Avis, les avis avec le statut "Affiché" apparaissent sur votre site. Les avis "En modération" ou "Masqués" restent invisibles pour vos visiteurs. Vous gardez le contrôle total.

Votre lien de collecte

Partagez ce lien avec vos clients pour collecter des avis. Chaque soumission arrive directement dans votre tableau de bord pour modération.

Template email à envoyer

Envoyez ce message à vos clients après une prestation :

Bonjour [Prénom], Je me permets de vous contacter suite à notre collaboration. Votre retour est très précieux pour moi — pourriez-vous prendre 2 minutes pour laisser un avis ? 👉 Lien de collecte Merci beaucoup pour votre confiance, [Votre prénom]

Partager sur WhatsApp / SMS

Message court idéal pour WhatsApp ou SMS :

Bonjour [Prénom] 👋 Merci pour votre confiance ! Pourriez-vous laisser un petit avis ici ? Ça prend 1 min : Lien de collecte

QR Code de collecte

Imprimez ce QR code sur vos supports (cartes de visite, devis, factures, flyers…). Vos clients le scannent et laissent un avis en quelques secondes.

Comment l'utiliser ?

Téléchargez l'image et intégrez-la dans vos documents Word, PDF ou présentations. Vous pouvez aussi l'imprimer directement et la coller sur vos supports physiques.

Le QR code pointe vers votre page de collecte personnalisée — chaque scan est tracé dans votre tableau de bord.
